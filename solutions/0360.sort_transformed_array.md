# 360.Sort Transformed Array

### LeetCode 题目链接

[360.Sort Transformed Array](https://leetcode.cn/problems/sort-transformed-array/)

### 题目大意

给一个已经`排好序`的整数数组 `nums` 和整数 `a` 、`b` 、`c`

对于数组中的每一个元素 `nums[i]`，计算函数值 `f(x) = ax^2 + bx + c`，请按`升序`返回数组

```js
Example 1:
Input: nums = [-4,-2,2,4], a = 1, b = 3, c = 5
Output: [3,9,15,33]

Example 2:
Input: nums = [-4,-2,2,4], a = -1, b = 3, c = 5
Output: [-23,-5,1,7]
```

说明:
- 1 <= nums.length <= 200
- -100 <= nums[i], a, b, c <= 100
- nums is sorted in ascending order.

### 解题

```java
class Solution {
    public int[] sortTransformedArray(int[] nums, int a, int b, int c) {
        int len = nums.length;
        int i = 0, j = len - 1;
        // 开口朝上，越靠近对称轴函数值越小
        // 开口朝下，越靠近对称轴函数值越大
        int p = a > 0 ? len - 1 : 0;
        int[] res = new int[len];
        while(i <= j) {
            int v1 = f(nums[i], a, b, c);
            int v2 = f(nums[j], a, b, c);
            // 开口朝上，越靠近对称轴函数值越小
            if (a > 0) {
                if (v1 > v2) {
                    res[p--] = v1;
                    i++;
                } else {
                    res[p--] = v2;
                    j--;
                }
            } else {
                // 开口朝下，越靠近对称轴函数值越大
                if (v1 > v2) {
                    res[p++] = v2;
                    j--;
                } else {
                    res[p++] = v1;
                    i++;
                }
            }
        }
        return res;
    }

    int f(int x, int a, int b, int c) {
        return a*x*x + b*x + c;
    }
}
```

```js
    let len = nums.length;
    let i = 0, j = len - 1;
    let p = a > 0 ? len - 1 : 0;
    let res = new Array(len);
    while(i <= j) {
        let v1 = f(nums[i], a, b, c);
        let v2 = f(nums[j], a, b, c);
        if (a > 0) {
            // 如果开口朝上，越靠近对称轴函数值越小
            if (v1 > v2) {
                res[p--] = v1;
                i++;
            } else {
                res[p--] = v2;
                j--;
            }
        } else {
            // 如果开口朝下，越靠近对称轴函数值越大
            if (v1 > v2) {
                res[p++] = v2;
                j--;
            } else {
                res[p++] = v1;
                i++;
            }
        }
    }
    return res;
};

var f = function(x, a, b, c) {
    return a*x*x + b*x + c;
};
```
